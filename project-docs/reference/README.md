# Reference Docs

This folder contains the migrated core reference documents that previously lived under `apps/smart_accounting/docs/`.

## Files

- `A_Data_Model_Assessment.md`
  Data model, field design, and structural decisions.
- `B_Code_Architecture_Review.md`
  Code architecture review and refactoring direction.
- `C_Business_Process_Flows.md`
  Workflow and process reference.
- `D_UI_Design.md`
  UI structure, interaction design, and visual decisions.
- `E_Implementation_Tutorial.md`
  Implementation and setup guidance.
- `architecture.md`
  Architecture rules, module boundaries, and engineering conventions.

## Usage

- Use this folder for stable reference material.
- Use `../r-and-d-notes/` for weekly progress, validation, and ongoing working records.
- Use `../reference-library.md` when you want the grouped reading guide.
